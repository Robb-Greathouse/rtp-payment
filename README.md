## General info
This project is simple banking demo application for Open Banking.
	
## Technologies
Project is created with:
* Angular CLI: 9.1.7
* Node: 12.16.1
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../rpt-payment
$ npm install
$ npm start
```
